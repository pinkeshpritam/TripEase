1.# Project Title
       -TripEase - Cab Booking Backend Application
   
2.# Project Description
    - TripEase is a Spring Boot based backend application for managing cab booking services. 
      It allows users to register, book cabs, assign drivers, and manage ride details using REST APIs
      
3.# Features
     - User Registration and Login
     - Driver Management
     - Cab Management
     - Ride Booking
     - REST APIs for operations
     - Database integration using MySQL
     
 4.## Technologies Used
     - Java
     - Spring Boot
     - Spring Data JPA
     - Hibernate
     - MySQL
     - Maven
     - Postman (API Testing)
     
5.## Project Structure
     - Controller : Handles API requests
     - Service : Business logic
     - Repository : Database operations
     - Model : Entity classes
     - DTO : Request and Response objects
     
6.## How to Run    
    1. Clone the repository
    2. Open project in IntelliJ / Eclipse
    3. Configure MySQL database
    4. Update application.properties
    5. Run the Spring Boot application
    
7.## API Testing
     You can test the APIs using Postman.
